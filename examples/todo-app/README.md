# examples/todo-app

Placeholder demo app referenced by the upstream ClaudeMaxPower CI
workflow (`.github/workflows/ci.yml` → `Run Example Tests`).

This directory exists so the inherited CI step finds something to
collect. The lab itself does **not** depend on this app — its
canonical entry points are `apps/runner` and `apps/cli`. See
[`README.md`](../../README.md) at the repo root.

A future task may flesh this out into a proper demo for the
`/fix-issue` skill.
